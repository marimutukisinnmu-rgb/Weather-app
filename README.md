# いまの天気・暑さ指数

環境省のWBGT（暑さ指数）と、気象庁の府県天気予報JSONを一画面に表示するPythonデスクトップアプリです。

## 起動

WindowsのPowerShellで、このフォルダを開いて次を実行します。

```powershell
python weather_dashboard.py
```

追加パッケージは不要です。Pythonに標準で含まれるTkinter、`json`、`urllib`のみを使用しています。

## データの扱い

- 気象庁: `www.jma.go.jp/bosai/forecast/data/forecast/*.json` の府県天気予報。
- 環境省: 熱中症予防情報サイトの全国47地点のWBGT表示。
- `今のデータ.js`: 通信できない場合に表示する、保存時点のWBGT値。これは最新値ではありません。

画面右上で代表地点を切り替え、`更新`で再取得できます。

検索欄では、初回検索時に気象庁の地域マスタを読み込みます。全国の市区町村を漢字・ひらがな・カタカナ・半角カタカナで検索できます（例: `八王子`、`はちおうじ`、`ﾊﾁｵｳｼﾞ`）。市区町村のWBGTは、環境省の全国47地点から同じ都道府県の代表地点を表示します。

## アプリ版を作る

Windows版は作成済みです。`dist/いまの天気・暑さ指数.exe` をそのまま起動できます。Pythonのインストールは不要です。

macOS版の `.app` は、macOS上で作成します。ソース一式をMacへコピーして、ターミナルで次を実行してください。古い `.app` を使わず、現在の `weather_dashboard.py` から作り直してください。

```bash
zsh build_macos.command
```

このコマンドは `/usr/local/bin/python3` を使い、初回だけ `bild` という仮想環境を作成・有効化して、必要なPyInstallerをインストールします。2回目以降はダウンロードを省略してすぐビルドします。完成品は `dist/いまの天気・暑さ指数.app` です。初回起動時にmacOSが警告した場合は、Finderでアプリをcontrolクリックして「開く」を選びます。

macOSアプリが起動直後に失敗した場合は、白画面ではなくエラー内容を表示します。
